# Welcome to Tasteey

**Tasteey** is a modern, flexible, and high-performance UI component library built for React. It leverages the power of **Tailwind CSS** for styling and **Framer Motion** for silky smooth animations.

<Callout title="Ready to build?" type="info">
  Tasteey is designed to be copy-pasted directly into your project. You own the code.
</Callout>

## Getting Started

To start using Tasteey in your project, simply browse the components, copy the code, and paste it into your application. We recommend installing a few core dependencies to ensure everything works out of the box.

```package-install
npm install motion clsx tailwind-merge
```
