# Hero

The Hero component is used at the top of SaaS landing pages to communicate value clearly.

## Example

<Hero
  title="Build faster. Ship smarter."
  subtitle="Motion-first SaaS UI components for modern teams."
  primaryCta={{ label: "Get Started", href: "#" }}
  secondaryCta={{ label: "View Demo", href: "#" }}
/>

## Props

| Prop           | Type     | Required | Description                               |
| -------------- | -------- | -------- | ----------------------------------------- |
| `title`        | `string` | ✅       | Main heading text                         |
| `subtitle`     | `string` | ❌       | Supporting text below the heading         |
| `primaryCta`   | `object` | ❌       | Primary action button `{ label, href }`   |
| `secondaryCta` | `object` | ❌       | Secondary action button `{ label, href }` |
